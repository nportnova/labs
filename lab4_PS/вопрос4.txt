4.Просмотр структуры объектов.

для просмотра структуры объектов в PowerShell используется командлет Get-Member 

командлет Get-Member позволяет перечислить только те элементы объекта, которые являются его свойствами. для этого используется параметр MemberType со значением Properties:

чтобы перечислить свойства:
Get-Process| Get-Member -MemberType Property (свойства)

процессам  ОС  соответствуют  объекты,  имеющие  очень  много свойств, на экран же при работе командлета Get-Process выводятся лишь несколько из них 

Get-Member выводит список всех членов объекта:

свойства (Properties) - данные объекта (например, имя процесса, ID, использование памяти)
методы (Methods) - действия, которые можно выполнить с объектом (например, остановить процесс, перезапустить)
события (Events) и другие типы членов.
